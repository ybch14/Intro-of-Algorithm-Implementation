# Insert Sort

Insert Sort algorithm implementation.

## Task definition

Sort a array from small to large.

## Test sample

- [1, 5, 6, 3, 4, 5, 6, 2, 1, 0]
- Standard normal distributed random-initialized 1000000 array (```data.txt```)

## Output

```
0 1 1 2 3 4 5 5 6 6
Data loaded!
Insert sort: 160.829s.
Insert sort correct? Yes! :)
```
